Niemann Family Finances

A local-first family budgeting app. All data lives in your browser via localStorage — no server, no account, no tracking.

Quick start

Requires Node.js 20.x or newer.

npm install
npm run dev      # http://localhost:5173

For production:

npm run build    # writes ./dist
npm run preview  # serves the production build at http://localhost:4173

Moving to another computer

  1. Copy the project folder (skip node_modules / dist if you want to keep it small).
  2. On the new machine: npm install, then npm run dev.
  3. Move your data separately — open the app, ⚙ Settings → ⬇ Download JSON backup, carry the file over, then ⚙ Settings → ⬆ Restore from JSON.

The data is stored in browser localStorage (key nfm_v1), not in the project files, so the project copy and the data backup are two distinct steps.

Ship it as a desktop app (macOS)

For someone who just wants to double-click an icon — not run terminals — build the native macOS app:

npm run dist:mac

This produces:

  • release/Niemann Family Finances-1.0.0-arm64.dmg — drag-to-Applications installer
  • release/mac-arm64/Niemann Family Finances.app — the unpacked app for direct testing

Target is Apple Silicon (M1/M2/M3/M4). See MANUAL.md → "Installing the macOS app" for the full handoff workflow.

Stack

  • React 18 + TypeScript + Vite
  • Tailwind CSS
  • Zustand (state + 60-step undo)
  • Chart.js (react-chartjs-2) for the donut
  • jsPDF + jspdf-autotable for PDF reports
  • PapaParse for CSV import / export

Highlights

  • Inline-editable ledger with pay periods (weekly / bi-weekly / semi-monthly / monthly)
  • Per-period budgets, with subcategory support on Bills (and any other category you flag "Allow subs")
  • CSV import for bank, credit, and savings accounts with smart header detection (handles parens-style negatives, two-column Debit/Credit, and metadata blocks)
  • Self-improving auto-categorizer — every manual category change becomes a durable rule that influences future imports
  • Splits, recurring rules (2-year horizon), donut chart with click-to-filter
  • Full JSON backup/restore + per-period PDF reports + CSV export
